# Wanaemic Portfolio Web App

A modern responsive portfolio web app built for deployment from the current workspace.

## Included files

- `index.html` — responsive portfolio landing page
- `styles.css` — modern glassmorphic styling, mobile-first layout, and animations
- `script.js` — dynamic content loading, reveal animations, and form handling
- `manifest.webmanifest` — basic PWA metadata
- `netlify.toml` — deploy settings for Netlify

## Features

- Hero landing section with professional portfolio messaging
- Responsive grid layout for work and skills
- Animated skill bars and project cards
- Contact form with instant feedback
- Clean typography and modern visual design

## Deploy

1. Host as a static site on any provider (`Netlify`, `Vercel`, `GitHub Pages`, etc.)
2. If using Netlify, simply connect this repository and deploy. The `netlify.toml` config is ready.

## Notes

The site is designed to highlight the strongest professional facts from the CV add-on and emphasize a polished production-ready portfolio presence.
